# Mi sitio web

Sitio estático con traducción entre español e inglés, carrusel en el inicio y páginas de detalle para proyectos y entradas del blog.

Páginas:
- index.html
- about.html
- projects.html
- blog.html
- project-portfolio.html
- project-branding.html
- project-app.html
- blog-portfolio.html
- blog-design.html
- blog-translation.html

Cómo añadir más entradas al blog:
- Edita el array `blogPosts` en `script.js`.
- Añade una fecha `YYYY-MM-DD`, temas y texto en español e inglés.
- Crea una página de detalle nueva siguiendo el patrón de las existentes.

Qué personalizar primero:
- Las fotos del carrusel del inicio.
- Tu correo, LinkedIn y GitHub en `projects.html`.
- Los textos de proyectos y del blog.
- Tu avatar cartoon en `assets/avatar-cartoon.png`.

Cómo probarlo:
- Abre `index.html` en tu navegador.
- Cambia el idioma con ES/EN y navega entre páginas.
